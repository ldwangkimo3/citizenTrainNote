筆記：20221008麥迪克TCCCASM上課筆記
標題：交火下的救護
主題：MARCH 處理程序


排除(等待)現場威脅解除
威脅解除後才能進行救助
趁機將傷患移動到更安全的地方
//(不考慮脊椎預防措施)
//若有機會，在傷口的高位處上緊止血帶，第一時間阻止危及生命的出血((判斷))
//
檢查4個跡象：
1，是否有噴血或大量流血 
2，衣服是否有大面積血液浸濕 
3，地上有積血 
4，完全斷肢
//(傷患意識正常：反擊/掩蔽/自救)
//
是否建立防線，保持警戒，以確保安全((判斷))
//
從頭接近傷患，避免誤擊
判斷若意識不佳，應卸除他的武器並關上保險、移除通訊設備和敏感物品
//

MARCH 處理程序
//
M 大出血處理 Massive Hemorrhage
檢查未確認到的出血部位
5出血點檢查
胸腹/四肢/生殖器到股溝 檢查
根據出血部位，使用止血帶/胸封貼片/止血敷料對應處理
//找到確實的傷勢，調整止血帶位置((判斷))
//
(胸封貼片：手臂平壓，打開貼片，抹平，貼上)
//
A 暢通呼吸道 Airway Management
//
聽看聞數，確認呼吸情形
//
讓有意識傷者採取任何最能保護呼吸道的姿勢
提下巴或推下巴
//
R 保持呼吸 Respiration/Breathing
張力性氣胸
胸部開放性/吸入性傷口
監測脈搏及血氧飽和度
中度/嚴重創傷性腦損傷
//
C 保持體循環 Circulation
頭(洗頭手勢)/臉
5出血點檢查/四肢/生殖器到股溝/背後
重新檢查止血帶

H 預防失溫 Hypothermia prevention
保溫毯
擔架
